# 🗺️ ROADMAP_V1 — Career OS Release Timeline

This document maps the development phases and vision of the **Career Operating System (Career OS)** from our current 125-agent index to global platform runtimes.

---

## 📊 Current State

- **Total Live Agents:** 125 (fully compliant and tested).
- **Active Divisions:** 14 functional categories (including AI Engineering, Cloud, Cybersecurity, Open Source, Data, and DevRel).
- **Core Tooling:** Converters (`convert.py`, `convert.sh`) and installers (`install.sh`, `install.ps1`) for Cursor, Claude Code, Codex, and Gemini CLI.

---

## 🎯 Version 1.0 Milestones (The Platform Scale)

**Target Timeline: Q3/Q4 2026**
- **Sitemap Portal Launch:** Release the React/Next.js web portal containing sitemaps, advanced search indices, and developer leaderboards.
- **One-Click Installer CLI:** Publish the `codemyfyp-agents` CLI package on npm for easy remote additions (`npx codemyfyp-agents add <agent-id>`).
- **Community Marketplace Release:** Integrate verification badges (`official`, `verified`, `community`) and rating models.
- **Campus Ambassadors Rollout:** Partner with placement cells and launch ambassador chapters in tier-1 universities.

---

## 🚀 Version 2.0 Vision (The Desktop Engine)

**Target Timeline: 2027**
- **Tauri Desktop Application:** Distribute a desktop agent client container with localized agent browsers, offline search caches, and workspace configurations.
- **Visual Workflow Runner:** Node-based editor allowing developers to visually map multi-agent pipelines and trigger mock tests.
- **Adjacent Editor Tooling:** Add one-click integrations for GitHub Copilot, Windsurf, Aider, and OpenCode.
- **Blockchain Certifications:** Issue verified academic credentials for completing preparation tracks.

---

## 🔮 Version 3.0 Vision (The Global Network)

**Target Timeline: 2028 and Beyond**
- **Decentralized Agent Network:** An open P2P marketplace where developers deploy fine-tuned custom agent models.
- **Enterprise Cohort Portals:** White-labeled corporate and college Career OS dashboards, offering dynamic applicant matching.
- **Global Placement Integration:** Partnerships with recruitment portals to directly match graduates' verified roadmaps with open job roles.
