# LinkedIn Optimizer Guide

This document describes the tagline auditing metrics, headline structures, and profile keyword density rules.

---

## Overview

The LinkedIn Optimizer scans headlines and profile summaries to ensure they utilize professional branding patterns and maintain high visibility in recruiter search engines.

---

## Recruiter Visibility Index

Recruiter algorithms rely heavily on specific keyword matching. The optimizer measures profile signaling across three areas:

1. **Tagline Structure**: Recommends using pipes (`|`) to separate roles, stacks, and achievements (e.g., `Backend Architect | Distributed Systems | Shipped API engines serving 100k users`).
2. **Industry Keywords**: Audits profile copy for critical technical competencies matching the candidate's target roles.
3. **Storytelling Narrative**: Checks summary copy for descriptive value propositions rather than flat checklists.

---

## Features

### 1. Tagline Rewriter
Suggests optimized alternative headlines containing pipes, metrics, and primary stacks.

### 2. Narrative Suggestions
Extracts weaknesses (e.g. passive vocabulary, lack of project metrics) and outlines summary expansion suggestions.

### 3. Skills Registry
Highlights list of recommended technical skill tags to append to the profile skills section.
