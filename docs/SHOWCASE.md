# Career-Agents Community Showcase

This showcase highlights integrations, wrappers, and dashboards built by developers using the **Career-Agents** CLI, registries, and MCP servers.

---

## 👥 Who Uses Career-Agents?

### 🎓 Students
-   **Placement Preparation**: Leverage specialized interview coaches to run mock loops targeting internship placement interviews.
-   **Final-Year Checklists**: Score technical skills and retrieve mapped roadmap paths for role readiness.

### 💻 Developers
-   **Direct IDE Prompts Injection**: Inject ATS checklists and interview rules directly into Cursor, Claude Code, and Windsurf workspaces.
-   **Role Pivoting**: Transition profiles (e.g. from Frontend to ML/AI Engineering) by retrieving target skill benchmarks.

### 💼 Career Coaches & Mentors
-   **Standardized Rubrics**: Use structured ATS resume rules to score candidate profile drafts and output consistent feedback.
-   **Structured Mock Loops**: Run diagnostic panels referencing target company prep tracks.

### 🚀 Founders & MVPs Scopers
-   **MVP Architectural Scoping**: Leverage specialized founder coaches to challenge product assumptions and draft software system requirements.
-   **Hiring Rubrics**: Construct technical competencies criteria matching target hire roles.

### 🔍 Recruiters & Talent Assessors
-   **ATS Keyword Benchmarks**: Extract core skills densities and compare candidates profiles against active job descriptions.

## 🛠️ Community Projects & Demos

### 1. Career-OS Desktop Dashboard
-   **Developer**: [Karthik Rajesh Shet](https://github.com/karthikrshet)
-   **Description**: A local desktop dashboard that parses `agent-registry.json` to launch agent sessions inside native frames.
-   **Code**: [GitHub Repo](https://github.com/karthikrshet/Career-Agents)

### 2. VS Code Career Agent Loader
-   **Description**: A VS Code extension adding a sidebar list of all divisions. Clicking an agent copies its prompt directly to the clipboard.
-   **Status**: Live in VS Code Extension Marketplace.
