# Agent File Standard

This document restates the canonical agent file standard used across the repository.

Required frontmatter

```yaml
---
name:

color:
emoji:

---
```

Required sections (in order)

1. `## ЁЯза Your Identity & Memory`
2. `## ЁЯОп Your Core Mission`
3. `## ЁЯЪи Critical Rules You Must Follow`
4. `## ЁЯУЛ Technical Deliverables`
5. `## ЁЯФД Workflow Process`
6. `## ЁЯТн Communication Style`
7. `## ЁЯФД Learning & Memory`
8. `## ЁЯОп Success Metrics`
9. `## ЁЯЪА Advanced Capabilities`

Deliverable guidance

- Use clear headings and copy-ready templates.
- Prefer bullet lists, tables, and labeled sections for outputs.

Quality metrics

- Minimum 1,000 words; reviewer discretion applies.
- Unique scope compared to existing agents.
