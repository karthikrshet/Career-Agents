# Community Growth Engine Specification

This document details the community engagement programs, discussion guidelines, and contribution leaderboards for the **Career-Agents** platform.

---

## 🚀 Key Programs

We drive community scaling through 5 core engagement models:

### 1. Developer Discussions
- Configured in the GitHub Discussions portal.
- Includes distinct directories for general support, Q&A, and showcasing custom agent configurations.

### 2. Custom Agent Requests
- Structured template in GitHub Issues allowing developers to suggest new target profiles.
- Approved requests are marked with the `help-wanted` and `new-agent` labels to recruit community developers.

### 3. Contributor Leaderboard
- A gamified dashboard on the website that ranks developers based on:
  - Total agents successfully merged (checked by `validate.py`).
  - Total issues resolved.
  - Pull requests accepted.
- Scores are updated dynamically via a GitHub Actions integration webhook.

### 4. Agent Sprints & Challenges
- Monthly maintenance sprints focused on writing agents for empty paths in expansion divisions (e.g. AI Engineering or Cybersecurity).
- Contributors receive custom profile badges and highlighted marketplace listings.

### 5. Career OS Hackathons
- Structured virtual hackathons challenging developers to build custom desktop integrations, workflow orchestration scripts, or local CLI plugins.
- Winners receive community sponsorships and official "Verified Partner" status.
