### Contribution Standards for AI
AI-assisted contributions are treated with the same high quality standards as human-authored code:
1. **Mandatory Validation**: Every PR must pass all checks. Run `python scripts/generate-data.py` and `python scripts/validate.py` before committing.
2. **Schema & Registry Compliance**: No duplicate IDs or orphaned agent files are allowed.
3. **No Hallucinated Paths**: Ensure that all generated directories, links, and config mapping boundaries are maintained.
