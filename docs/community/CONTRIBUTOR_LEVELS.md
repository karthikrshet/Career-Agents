# Contributor Levels

We recognize and reward developers, writers, and organizers who contribute to the growth of **Career-Agents**, the Open-Source Career Operating System. 

Our contribution system is structured into five recognition tiers:

---

## 🎖️ Contributor Tiers

### 1. Level 1: Scout
- **Requirements**: Filer of 2+ bug reports in CLI or registry syntax, or minor spelling/formatting fixes in agent prompts.
- **Perks**: Listed in standard repository contributor metrics; invitation to the Discord server.

### 2. Level 2: Builder
- **Requirements**: Contributed a complete new integration guide, workflow markdown, or CLI testing scripts.
- **Perks**: Badge added on GitHub profiles; listed in [HALL_OF_FAME.md](./HALL_OF_FAME.md).

### 3. Level 3: Strategist
- **Requirements**: Authored/Optimized 5+ high-fidelity agent prompt markdown files adhering to the gold-standard specification.
- **Perks**: Core voice in roadmap planning meetings; custom role on Discord.

### 4. Level 4: Ambassador
- **Requirements**: Lead local regional communities or organize campus hackathons introducing developers to Career-Agents.
- **Perks**: Mapped in [AMBASSADOR_PROGRAM.md](./AMBASSADOR_PROGRAM.md); sponsorship/swag access.

### 5. Level 5: Code Steward (Maintainer)
- **Requirements**: Nominated by existing maintainers after sustained leadership contributions, PR reviews, and releases coordination.
- **Perks**: Write/Merge access to the repository; listed in [MAINTAINERS.md](./MAINTAINERS.md).
