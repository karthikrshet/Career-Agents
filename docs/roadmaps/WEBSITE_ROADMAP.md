# Career-Agents Website Roadmap

This roadmap details the structure, future pages, and SEO architecture for the official **Career-Agents** web application.

---

## 🗺️ Future Page Sections

### 1. Home / Landing Page
-   **Hero taglines**: Interactive command line interface simulations, stats counters (agents counts, downloads, github stars).
-   **Products Overview**: Visual cards pointing to Resume Scorer, MCP Server and Division catalogs.

### 2. Agents Hub (`/agents`)
-   **Catalog Browser**: Interactive filters matching specialty divisions (e.g. AI-Engineering, Engineering, Startup).
-   **Prompt Viewer**: Online viewer allowing copy-to-clipboard or launcher triggers.

### 3. ATS Resume Studio (`/resume`)
-   **Visual ATS Scorer**: File uploader (JSON/txt/pdf) executing scoring checks in-browser.
-   **Templates Builder**: Configurator rendering the 20 resume studio templates layouts.

### 4. Company Prep Tracks (`/companies`)
-   **Target Tracks**: Detailed preparation timelines, skills lists, and associate coaching agents for tier-1 companies (Google, Stripe, Meta).

### 5. Career Paths (`/paths`)
-   **Interactive Roadmap**: Tree hierarchies detailing software engineer, data analyst, and PM milestones.

### 6. Relational Knowledge Graph (`/graph`)
-   **3D Network Visualizer**: Interactive d3.js network showing node connections linking divisions, skills, companies, and agents.

### 7. Documentation Portal (`/docs`)
-   **Unified Resource Center**: Consolidated integration guides and command references.

### 8. Sponsor & Support (`/sponsor`)
-   **Sponsorship Tiers**: Stripe integration for sponsor tiers checkout, listing Silver and Gold backers.
