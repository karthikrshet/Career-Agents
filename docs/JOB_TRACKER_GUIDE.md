# Job Tracker Guide

This document describes the job applications funnel, pipelines tracker, and status lifecycle mappings.

---

## Overview

The Job Application Tracker organizes target vacancies, interview milestones, and offer metrics.

---

## Funnel Stages

The system tracks jobs across five distinct lifecycle status categories:

1. **Wishlist**: Pre-referral target companies and role titles.
2. **OA Pending**: Online assessments assigned by recruiters.
3. **Interview Scheduled**: Technical screening, system design, or onsite loops.
4. **Offer**: Written employment offers received.
5. **Rejected**: Unsuccessful application runs.

---

## Metadata Fields

Each application entry stores:
- **Company**: Target name.
- **Role**: Position title.
- **Salary**: Annual compensation estimation.
- **Location**: Workplace details (e.g. Remote, City).
- **Notes**: Loop preparation status and contacts details.
- **Date**: Chronological added timeline.

---

## System Integration

The dashboard reads from the Job Tracker to display pipeline totals. Candidates can review active application stats from the main overview panels.
