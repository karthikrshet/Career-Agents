# Release Notes v2.0.0

This release marks the transition of Career-Agents into a fully operational **Career Operating System**.

## Highlights

- **Resume Studio**: Robust parsing (JSON, MD, TXT, and PDF) with section completeness audits and weak bullets optimization suggestions.
- **GitHub Analyzer**: Stars, forks, language diversity, and README quality grading index.
- **LinkedIn Auditor**: Copy-paste tagline hooks review and AI headline alternatives.
- **Interactive Interview Simulator**: Interactive terminal mock interview loops.
- **Ecosystem Dashboard**: Progress tracking dashboard.
- **Plugin Architecture**: Extend CLI commands dynamically.
- **Opt-in Telemetry**: Option to log anonymous usage events.
