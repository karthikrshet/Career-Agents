## 📥 Installation

Install globally via NPM:
```bash
npm install -g career-agents
```

---

## 🚀 Available Global CLI Commands

Ensure your global path environment variables are active, then run:

### 1. Doctor (Self-Diagnostics)
```bash
career-agents doctor
```

### 2. List Specialties
```bash
career-agents list
```

### 3. Assess Career Strategy Score Card
```bash
career-agents assess
```

### 4. Recommend Specialized Agents
```bash
career-agents recommend
```

### 5. Inspect Company tracks
```bash
career-agents company google
```

### 6. Inspect Career Mapped Milestones
```bash
career-agents path ai-engineer
```

### 7. Inspect Resume Templates
```bash
career-agents resume templates
```
