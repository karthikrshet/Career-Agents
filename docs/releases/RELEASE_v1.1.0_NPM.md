# Career-Agents v1.1.0 — Official NPM Release

We are excited to announce the official publication of **Career-Agents** to the NPM registry! The Open Source Career Operating System is now globally installable via command line portals.

Tag: `v1.1.0`

---

## 🚀 Release Highlights

-   **✅ Published to npm**: Easily install globally via the NPM registry.
-   **✅ ATS Resume Studio**: Structure audits, layout checks, and templates mapping for 20 engineering templates.
-   **✅ Career CLI**: Interactive self-diagnostic tools (`doctor`), profile recommendation reports, and strategy scoring questionnaire boards.
-   **✅ MCP Server**: Full editor compatibility (Cursor, Claude, Windsurf, OpenCode) with 10 tools and 7 resource registries.
-   **✅ Career Paths**: Sequenced milestones and competency plans.
-   **✅ Company Interview Tracks**: Target interview structures and preparation resources for tier-1 tech firms.
-   **✅ Resume Intelligence**: Dynamic extraction of skills, formatting, and formatting gaps.
-   **✅ Job Match Intelligence**: Native keyword mapping algorithms auditing resume-to-job matching percentages.
-   **✅ Knowledge Graph**: Network graph coordinate maps of 400+ nodes linking paths, skills, companies, and coaches.

---

## 📥 Installation

```bash
npm install -g career-agents
```
