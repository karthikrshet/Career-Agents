# Showcase Examples Index

This document maps the locations and descriptions of the sample profiles, audits, and configuration files provided in the `examples/` directory.

---

## Examples Manifest

### 1. examples/resume.md
- **Description**: Sample candidate resume in Markdown format. Structuring achievements, skills sections, and project descriptions.
- **Path**: `examples/resume.md`

### 2. examples/github-analysis.json
- **Description**: Sample JSON output compiled by the GitHub Analyzer. Contains star counts, fork counts, language details, and README grades.
- **Path**: `examples/github-analysis.json`

### 3. examples/linkedin-analysis.md
- **Description**: Sample audit report compiled by the LinkedIn Optimizer. Contains tagline pipe (`|`) rewrites and profile summaries checklist.
- **Path**: `examples/linkedin-analysis.md`

### 4. examples/dashboard.json
- **Description**: Sample serialized user profile state. Links resume score, GitHub stats, LinkedIn index, and interview loops grade.
- **Path**: `examples/dashboard.json`

### 5. examples/report.html
- **Description**: Sample visual HTML report card layout template. Incorporates dashboard dials and audit comments.
- **Path**: `examples/report.html`

### 6. examples/report.md
- **Description**: Standard Markdown report card template format. Used to export console audits.
- **Path**: `examples/report.md`
