# Career-Agents Community Growth Plan 2026

This roadmap details key objectives, execution milestones, and metric targets for expanding the **Career-Agents** ecosystem.

---

## 🎯 Target Milestones (2026)

-   **⭐ 100 GitHub Stars**: Build organic developer awareness.
-   **👥 10 Contributors**: Establish active maintainers and community contributors.
-   **📥 1,000 NPM Downloads**: Grow global command line package installations.
-   **💖 10 GitHub Sponsors**: Support ongoing operational and community workshop budgets.

---

## 🗓️ 30-Day Execution Plan (Immediate Focus)

### 1. GitHub Stars & Discovery
-   Submit Career-Agents to Hacker News (Show HN) and Product Hunt.
-   Publish a technical launch post on DEV.to and Hashnode explaining how the Model Context Protocol (MCP) server integrates into Cursor and Claude Desktop.

### 2. NPM Downloads
-   Add NPM install commands as recommendations in related dev community groups.
-   Promote the `doctor` command to ensure setup is extremely reliable.

---

## 🗓️ 90-Day Execution Plan (Medium-Term Expansion)

### 1. Developer Engagement & Contributors
-   Create a "good first issue" tag labeling 10+ entry-level tasks (e.g. adding new company interview tracks, fixing typos in prompts).
-   Publish a formal [CONTRIBUTING.md](./CONTRIBUTING.md) detailing local development setup.

### 2. Sponsorships
-   Publish the first community newsletter showing how backer funding is spent.
-   Provide direct perks (e.g. priority mock interview signups) for early sponsors.

---

## 🗓️ 12-Month Execution Plan (Long-Term Scaling)

### 1. Ecosystem Integration
-   Build a browser-based web dashboard displaying agent catalogs.
-   Collaborate with university placement teams to adopt Career-Agents as a tool for students.

### 2. Community Events
-   Establish monthly peer-to-peer hackathons to write new specialized career agent prompts.
-   Host mock interview preparation loops on Discord twice a week.
