# Projects Showcase

This showcase highlights tools, workflows, wrappers, and dashboards built by the community using the **Career-Agents** registries, CLI, or indices.

---

## 🛠️ Applications & Tools

### 1. Career-OS Desktop Dashboard
- **Developer**: [Karthik Rajesh Shet](https://github.com/karthikrshet)
- **Description**: A local desktop dashboard that parses `agent-registry.json` to let users launch agent sessions inside local web views.
- **Link**: [CodeMyFYP-Agents Web App](https://github.com/karthikrshet/Career-Agents)

### 2. Auto-Applier Workflow Runner
- **Description**: A terminal runner script that fetches target roles from `career-paths.json` and automatically triggers matching job description scrapers.
- **Source**: Community Contributed wrapper.

---

## 🔌 IDE Extensions

### 1. VS Code Career Agent Loader
- **Description**: VS Code extension that adds a sidebar list of all divisions. Clicking an agent automatically writes it to the local workspace's `.cursorrules` or `.github/copilot-instructions.md`.
- **Status**: Live in VS Code Marketplace.
