# Community Guide

Welcome! This guide provides a step-by-step onboarding walkthrough for new members joining the **Career Operating System** community.

---

## 🚀 Onboarding Walkthrough

### 1. Join Our Discord
- Introduce yourself in the `#introductions` channel.
- Set your developer tags: e.g. Frontend, Backend, AI, Student, Founder.

### 2. Configure Local Tooling
- Clone the repository and execute `npm install`.
- Run:
  ```bash
  npx career-agents doctor
  ```
  Ensure all configurations are green.

### 3. Join Saturday Mock Panels
- Saturday mock loops are coordinated in the `#mock-interviews` channel on Discord.
- Read through the guidelines in `interview/mock-interviewer.md`.

---

## 🤝 Code of Conduct
We are committed to maintaining a welcoming and supportive environment. Please respect the standards outlined in [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md) during all chats, code reviews, and meetings.
