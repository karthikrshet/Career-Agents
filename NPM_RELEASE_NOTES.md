# Career-Agents v1.0.0 Release Notes

Welcome to the official **v1.0.0** release of Career-Agents, a production-ready, globally installable developer tool and MCP-powered Career Operating System.

---

## 🚀 Release Highlights

-   **135+ Specialized AI Agents**: Ready-to-use personas covering Technical Interviews, System Design, Career Strategy, MLOps, DevRel, and Startup Strategy.
-   **ATS Resume Studio**: Audit formatting, analyze keywords density, and score ATS compliance for 20 engineering and PM resume templates.
-   **Model Context Protocol (MCP) Server**: Full integration with AI coding assistants (Claude Desktop, Cursor, Windsurf, OpenCode) with custom tools and resource loaders.
-   **Global CLI**: Zero-dependency command utility (`career-agents`) supporting diagnostics, catalog searches, recommendations, and assessments.
-   **Company Interview Tracks**: Pre-packaged preparation playbooks and interview loops for tier-1 companies (Google, Stripe, Meta, Netflix).
-   **Structured Career Paths**: Sequence roadmap milestones, milestones training plans, and key competency requirements.
-   **Expanded Knowledge Graph**: Relational catalog maps containing 400+ nodes and 1700+ connections linking skills, agents, and paths.
-   **Resume & Job Match Intelligence**: Native keyword extracting algorithms calculating resume-to-job matching percentages and missing keywords.
