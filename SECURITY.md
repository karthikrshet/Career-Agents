# Security Policy

## Reporting a Vulnerability

If you discover a security issue in CodeMyFYP-Agents, please report it privately by opening an issue with the label `security` or contacting the maintainers directly if contact information is available.

Do not include sensitive details in a public issue if they expose a vulnerability.

## Supported Versions

Security fixes will be addressed for the current repository version and the two previous minor release versions.

## Response Process

1. Acknowledge receipt within 48 hours.
2. Assess the report and classify severity.
3. Notify maintainers and affected contributors.
4. Publish a fix and update release notes once verified.
