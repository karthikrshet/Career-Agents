# Generated Agent Prompt Exports

This directory holds the generated configuration payloads and prompt bundles emitted by the Career-Agents CLI.

To export any agent, run:
```bash
career-agents run <agent-id> --export <format>
```

Supported formats:
- `markdown` / `md`
- `json`
- `yaml`
- `txt`
- `prompt-bundle`
- `llm-bundle`
