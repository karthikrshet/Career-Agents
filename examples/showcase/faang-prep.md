# Showcase: FAANG Preparation Case Study

This showcase documents the multi-week preparation timeline utilized by a candidate to secure multiple tier-1 software engineering offers.

---

## 🧑‍💼 Candidate Persona
- **Role**: Senior Infrastructure Engineer (Go, Kubernetes)
- **Goal**: Clear Google and Meta Systems Design + Coding tracks.

---

## 🧭 The 4-Week Prep Blueprint

```
[Week 1: Foundations] ──> [Week 2: System Design] ──> [Week 3: Leadership] ──> [Week 4: Mock Drills]
  - Run Recommender          - system-design-coach       - behavioral coach         - mock-interviewer
  - Pick FAANG bundle        - AWS Cloud Architect       - leadership-coach         - company-specific
```

### Week 1: Diagnostics & Alignment
- **CLI Commands**:
  - `career-agents bundles faang-bundle` (Analyze prep checklist).
  - `career-agents score` (Identify gaps).
- **Deliverables**: Drafted 6 STAR stories mapping architecture scale achievements.

### Week 2: Deep-Dive System Architectures
- **CLI Commands**:
  - `career-agents run system-design-coach` (Practice requirements scaling).
  - `career-agents run kubernetes-specialist` (Review cluster configurations).

### Week 3: Behavioral Leadership
- **CLI Commands**:
  - `career-agents run leadership-interview-coach` (Study organizational conflicts resolution).
  - `career-agents run behavioral-interview-specialist` (Refine narrative structure).

### Week 4: Company-Specific Tracks
- **CLI Commands**:
  - `career-agents company google` (Inspect Google coding expectations).
  - `career-agents run google-interview-coach` (Live simulated roleplay).
  - `career-agents run meta-interview-coach` (Meta fast execution loops).

---

## 📈 Out-of-the-Box Results
- **Offer Metrics**: Cleared Google SWE L5 and Meta Production Engineer L5 loops simultaneously.
