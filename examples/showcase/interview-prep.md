# Showcase: Interview Preparation Case Study

This showcase illustrates how a candidate structured their weekly technical mock rounds to prepare for high-pressure system design and coding screens.

---

## 🧑‍💼 Candidate Persona
- **Role**: DevOps / Site Reliability Engineer (SRE)
- **Goal**: Clear systems architectural debugging and high-availability design interviews.

---

## 📅 The 7-Day Prep Routine

| Day | Focus Area | Active CLI Agent |
| :--- | :--- | :--- |
| **Day 1** | Architectural Diagnostics | `career-agents run backend-architect` |
| **Day 2** | Database Scaling Bottlenecks | `career-agents run database-performance-specialist` |
| **Day 3** | Mock System Design loops | `career-agents run system-design-coach` |
| **Day 4** | Behavioral Story Checklists | `career-agents run behavioral-interview-specialist` |
| **Day 5** | Live Mock Roleplay Simulation | `career-agents run mock-interviewer` |
| **Day 6** | Communication Rubric Review | `career-agents run group-discussion-coach` |
| **Day 7** | Final Review & Compensation Sync | `career-agents run salary-negotiation-coach` |

---

## 🏆 Sample Evaluation Scorecard
Candidate scorecards logged by the `Mock Interviewer` evaluate performance across:
1. **Requirements Gathering** (functional vs non-functional constraints definition).
2. **Component Architecture** (clear caching, databases, scaling parameters).
3. **Bottleneck Analysis** (mitigating single points of failure).
4. **Communication Vibe** (structured structured speech, taking hints constructively).
