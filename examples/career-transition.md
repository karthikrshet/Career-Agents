# Career Transition Example: Non-Technical to Full Stack Engineer

This guide demonstrates how to execute a transition pivot using the Career OS bundles and path engines.

---

## 🗺️ Step 1: Analyze Path Engine
Run the path engine to identify core competencies and recommended tools:
```bash
node scripts/cli.js path software-engineer
```
Observe the required skills: CS Fundamentals, Object Oriented Design, System Design, Git.

## 🎁 Step 2: Deploy Career Switcher Bundle
Activate the pre-packaged pivot assets:
```bash
node scripts/cli.js bundles career-switch-bundle
```
This maps the optimal path sequence:
- `career-pivot-to-tech-advisor` (agent): Selects technical entry points and constructs learning timelines.
- `career-transition-coach` (agent): Identifies transferable functional assets and bridges portfolio gaps.

## 👥 Step 3: Run Transition Advising Session
Launch the local interactive execution loop:
```bash
node scripts/cli.js run career-pivot-to-tech-advisor
```
Describe your current background (e.g. Sales, Operations, Marketing) and review the structural action plans.
