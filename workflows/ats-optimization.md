# ATS Optimization

## Objective
Turn your resume into an ATS-friendly application engine capable of passing automated filters and impressing hiring teams.

## Target audience
- Candidates whose applications are not converting to interviews.
- Job seekers applying through corporate portals and large hiring programs.
- Anyone who needs to improve ATS compatibility without sacrificing readability.

## Recommended agents
- `career/ats-resume-reviewer.md`
- `career/resume-strategist.md`
- `career/linkedin-growth-advisor.md`
- `career/job-search-strategist.md`
- `career/placement-coach.md`

## Step-by-step process
1. Run the resume through `ats-resume-reviewer` to diagnose parsing, keyword, and format issues.
2. Align role-specific keywords and phrases using `resume-strategist` without resorting to keyword stuffing.
3. Normalize heading structure, section order, and file format for ATS compatibility.
4. Validate with template tests, alternate file formats, and a human readability pass.
5. Confirm the resume still communicates achievements clearly to recruiters.
6. Use `job-search-strategist` to match applications to roles that value both ATS fit and human review.
7. Track response patterns and iterate on weak-performing variants.

## Weekly execution plan
Week 1: Audit the resume and fix all ATS red flags.
Week 2: Target 3 role families with role-specific keyword variants.
Week 3: Run live applications, collect feedback, and refine the resume based on actual results.

## Success metrics
- Percent improvement in application-to-interview conversion.
- Reduction in manual application rejections before screening.
- ATS parse success across multiple role types.
- Readability score for both machines and humans.
- Number of interviews generated from tracked applications.

## Common mistakes
- Designing a resume purely for ATS while making it unreadable for hiring managers.
- Relying on generic keyword lists instead of role-specific phrases.
- Ignoring the importance of a strong summary and impact bullets.
- Failing to test the resume across different ATS-friendly formats.

## Expected outcomes
- A resume that passes ATS filters more consistently.
- A clearer signal that your experience matches targeted roles.
- Higher interview rates from role-specific applications.
- A scalable resume versioning system for future roles.
