# Career-Agents Sponsorship Program

Help support the ongoing development of the **Open-Source Career Operating System**. Sponsoring Career-Agents helps keep prompt databases synchronized, maintains CLI/MCP compatibilities, and supports community initiatives.

---

## 💖 Why Sponsor Career-Agents?

As an independent, open-source project, we rely on backing from developers and organizations to keep building. Your support pays directly for:
-   Updating specialized AI agent prompt files matching current industry standards.
-   Adding new company preparation tracks and career path files.
-   Developing more IDE integrations (e.g. VSCode, JetBrains).
-   Running weekly mock panels and hiring workshops.

---

## 🗺️ Project Funding Roadmap

-   **$500 / month**: Maintain prompt files, upgrade MCP schemas, and fix bugs.
-   **$1,500 / month**: Launch new company-specific tracks and engineering roadmaps.
-   **$3,000 / month**: Build official web applications and interactive prompt editors.
-   **$5,000 / month**: Finance hiring workshops, mock panels, and resume reviews for community members.
